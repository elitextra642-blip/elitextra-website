# Deployment Notes

## Fastest Publish Path

Use a static host for the first public version. The site does not require a build command.

Recommended first publish:

- Cloudflare Pages or Netlify.
- Build command: leave empty.
- Publish directory: repository root.

## Before Sharing With Clients

- Confirm the public domain.
- Confirm the inquiry email address.
- Confirm whether the form should connect to Netlify Forms, Formspree, EmailJS, Supabase, or another backend.
- Confirm legal links or temporarily point them to contact until final legal copy exists.
- Confirm any claims about savings, certifications, markets, and export capacity.

## After Publish

- Update `sitemap.xml` and `robots.txt` if the domain is not `https://elitextra.com`.
- Add a social preview image.
- Run a mobile check on the live URL.
- Test every CTA from home, products, quote, and contact.
