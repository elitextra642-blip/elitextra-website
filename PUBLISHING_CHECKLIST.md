# Elitextra Publishing Checklist

## First Public Version

The first publishable version does not need every 3D/video idea finished. It must look credible, image-rich, communicate clearly, and give buyers a direct inquiry path while we continue toward the full dynamic platform.

## Must Be Ready Before Publishing

- Homepage has strong buyer-focused positioning.
- Product catalogue shows all current product lines clearly.
- Quote page works as a structured inquiry flow.
- Contact page does not include fake or unverified contact details.
- Logo is consistent across pages.
- All major images load from local `assets/` paths.
- Mobile layout is readable and usable.
- Forms do not falsely claim server submission until a real backend is connected.
- Sitemap and robots files exist.
- No `_archive/` files are included in the deploy output.

## Should Be Improved Before Client Showcase

- Replace weak or overly artificial hero visuals with stronger local campaign imagery.
- Add more product-specific copy from flyers and reference images.
- Add market-specific value sections for USA, UK, Europe, and Nigeria.
- Add quality/compliance content if certifications are confirmed.
- Add a real form destination: Formspree, Netlify Forms, EmailJS, Supabase, or a custom backend.
- Add Open Graph preview image for social sharing.

## Dynamic Upgrade Candidates

- Animated product range carousel or scroll stack.
- Interactive buyer path planner.
- Farm-to-port sourcing route animation.
- Product detail pages with richer specifications.
- Background video loops for hero and sourcing sections.
- Optional Spline or Three.js product cluster if assets are available.

## Do Not Publish Until Confirmed

- Pricing claims.
- Certification claims.
- Phone/WhatsApp number.
- Exact farm/cooperative names.
- Export volumes or buyer counts.
- Legal policy language.

## Deployment Options

- Cloudflare Pages for fast first publish.
- Netlify for easy form handling.
- Vercel if the site migrates to Next.js.
- GitHub Pages for a simple temporary presence.
